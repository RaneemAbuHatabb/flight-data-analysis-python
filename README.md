# Flight Data Analysis using Python

## Project Overview

This project analyzes flight records using Python and performs exploratory data analysis (EDA) to discover trends, airline performance, and flight patterns.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Analysis Performed

* Most common airline carrier
* Most frequently used aircraft
* Flight distribution by time of day
* Monthly flight trends
* Weekday vs weekend flight comparison
* Airline departure delay analysis

## Key Skills Demonstrated

* Data Cleaning
* Data Analysis
* Exploratory Data Analysis (EDA)
* Data Visualization
* Python Programming
* Problem Solving
